# Installation

### Install Go

MetaOS is built using [Go](https://golang.org/dl/) version 1.17+

### Install Binaries

#### Github

Clone and build MetaOS using `git`:

```bash
git clone https://github.com/metaos-labs/metaos
cd metaos
make install
```

Check that the `metaosd` binaries have been successfully installed:

```bash
metaosd version
```
